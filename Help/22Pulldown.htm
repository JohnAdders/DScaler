2:2 Pulldown Settings
 

2:2 Pulldown detection uses the comb factor measurement to determine whether the material was sourced from film or from video. The default settings have been found to work with a wide range of inputs but there are problems with some highly detailed images.

Setting Default Description
Pulldown Threshold Low 40 This is the percentage drop in comb factor that will be considered to be a candidate film mode.
Pulldown Threshold High 10 This is the percentage increase in comb-factor that will be considered evedence that we are no longer in film mode
PAL Video Mode Dependent on CPU selection in general hardware dialog box This is the mode that will be used when film mode has not been detected
PAL Pulldown Repeat Count In 3 Number of times film mode must be spotted in a row before a switch to film mode
PAL Pulldown Repeat Count Out 1 Each time bad pulldown is encountered the count is decremented starting at "PAL Pulldown Repeat Count In", once the count reaches this value then we go back to video mode.
Max Calls to PAL Comb Method 20 If we have tuned off "Bad Pulldown Filter" then this is the number of calls allowed to the bad cadence method before switching back to video mode.
PAL Bad Cadence Mode Greedy Low Motion If we have tuned off "Bad Pulldown Filter" then this mode is activated when bad cadence is detected.
