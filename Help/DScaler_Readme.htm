Requirements:

  • I think it will run on Win 95/98 and NT. I have tested it on 98, ME and Windows 2000.
  • You will need a processor faster than about Pentium II 400MHz to use all the features.
  • You will definitely need a PCI TV card with a BT848 BT849 BT878 BT879 CX23880 CX23881 CX23883 or SAA7134 chip in it.
  • You will a graphics card that supports YUY2 overlays and has enough video memory spare for the video buffers DScaler uses.
  • This software will not work with the video in sections of ATI All-in-Wonder or Matrox TV cards. These multifunction cards are supported for output though.

Increased Processor Requirements

Some of the improvements to film detect since version 2.3.0 have increased the processor requirements. DScaler may not now run on slower hardware.

Warning

This program installs a temporary driver while it runs.  This means that some of the code runs inside Ring0 along with the kernel.  What this means is that if anything goes wrong this program can crash your whole system causing a BSOD in NT or a hanging 95/98.  Please do not use kill task to stop DScaler except as a last resort.

If using a debugger then it it is very important that the program is allowed to shut down normally.  If you use Stop Debugging while running the program you system WILL crash.

As a precaution sync the disks before running this program.  Use the sync utility from http://www.sysinternals.com/ for whichever operating system you are running.
