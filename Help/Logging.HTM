Logging Settings
 

DScaler has an log feature that is useful for collecting infomation to attach to bug reports or for developement. Under normal circumstances logging should be off. When switched on infomation is written to DScaler.txt in the DScaler directory.

Setting Default Description
Debug Log OFF Whether logging is on or off.
Debug Level 1 The anount of logging to produce, higher values give more detailed output.
Debug Log File Name DScaler.txt The file to use to write logging.
