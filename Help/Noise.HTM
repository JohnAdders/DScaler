Temporal Noise Settings
 

Temporal noise reduction filter. This noise filter smooths out slight variations in pixels between adjacent frames. If the difference between a pixel and its predecessor from two fields back is less than a threshold, replace the pixel on the current field with a weighted average of its current and previous values.

Setting Default Description
Temporal Luminance Threshold 6 Pixel luminance differences below this are considered noise.
Temporal Chroma Threshold 7 Pixel chroma differences below this are considered noise.
Fast Memory Access On This speeds up the filter, but is incompatible with some VIA motherboards.
