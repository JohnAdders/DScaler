Overlay Settings
 

Setting specific to the Overlay

Setting Default Description
Back Buffers -1 Number of back buffers to use. -1 means get maximun that card will support, usually 2
Overlay Colorkey RGB(32,16,16) Color to use for the overlay colorkey.
Use Overlay Controls OFF Should we control the Graphic card's overlay controls.
Overlay Brightness 75 Allows setting of the graphic card's overlay Brightness
Overlay Contrast 100 Allows setting of the graphic card's overlay Contrast
Overlay Hue 0 Allows setting of the graphic card's overlay Hue
Overlay Saturation 100 Allows setting of the graphic card's overlay Saturation
Overlay Gamma 1 Allows setting of the graphic card's overlay Gamma
Overlay Sharpness 5 Allows setting of the graphic card's overlay Sharpness.
