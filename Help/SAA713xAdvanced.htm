SAA713x Advanced Settings
 

Advanced Settings for SAA7130/4 based cards. The default values are chosen to work well with most situations and usually do not need to be changed.

Setting Default Description
Automatic Gain Control ON Enable or disables the card's automatic gain control.
Automatic Volume Leveling MEDIUM The card automatically changes the volume level to avoid digital clipping. Not supported on the SAA7130.
Custom Pixel Width 750 The pixel width used when Custom is selected as the Pixel Width.
Gain Control Level 256 The gain level used when Automatic Gain Control is switched off.
Horizontal Delay 0 Shifts the picture right or left horizontally.
Mirroring OFF Enables the card's mirroring filter.
Reverse Polarity OFF Switches odd and even field order.
VBI Debug Overlay OFF Overlays VBI data onto a portion of video so that the VBI data given by the card can be seen.
VBI Upscale Divisor 424 Used by the card to determine the VBI sampling rate. Lower means higher sampling rate, hence better VBI decoding. Not all cards can handle low values. VBI Debug Overlay can be used to aid adjusting.
Vertical Delay 0 Shifts the picture up or down vertically.
